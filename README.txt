How to run the project:

1- Create a schema in mysql with name: bank_schema
2- Run all the sql ststements in the MySQL_Script.sql file
3- Open Eclipse and import the project folder BM_System.
4- Change the details (username, password) in DB_Handler.java according to your settings.
4- Run the project


Note:
Login accounts for Manager and Accountant are already created in database through sql file.
Some data is also added through the same MYSQL_Script file.
